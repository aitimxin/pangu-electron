# 更新日志

## [1.0.0] - 2024-11-04

### 新增功能

- ✨ 初始版本发布
- ✨ 实现本地 Puppeteer 视频抓取功能
- ✨ 支持抖音、B站、快手平台
- ✨ 集成 ChatGPT 对话界面
- ✨ 实现系统托盘功能
- ✨ 实现全局快捷键
- ✨ 实现自动更新功能
- ✨ 实现配置管理
- ✨ 实现日志记录
- ✨ 支持对话记录导出

### 技术特性

- 🔧 基于 Electron 28
- 🔧 集成 Puppeteer 21
- 🔧 进程隔离和安全桥接
- 🔧 跨平台支持（Windows/macOS/Linux）

### 已知问题

- ⚠️ 首次启动需要下载 Chromium，时间较长
- ⚠️ 部分视频平台可能需要登录才能抓取

## 版本说明

版本号格式：主版本号.次版本号.修订号

- **主版本号**：重大架构变更或不兼容的 API 修改
- **次版本号**：新功能添加，向下兼容
- **修订号**：Bug 修复和小改进
